## 개요
RealTrainMod에 JR 동일본 EV-E301계 전동차를 추가하는 애드온 팩입니다.

## 동작 확인 환경
* Minecraft 1.7.10 (KaizPatchX-1.7.1)
* Minecraft 1.12.2 (RTM 2.4.23-42)

## 설치 방법
0. `mods` 폴더에서 `[RTM]SeriesEV-E301_KarasuyamaLine.zip`을 삭제합니다 (이전 버전이 설치된 경우).
1. 다운로드한 `[RTM]Kozu_JR-East_EV-E301Series-Pack.zip`을 `mods` 폴더에 복사합니다.
2. Minecraft를 실행하여 각 모델이 추가되었는지 확인합니다.

### 필수 팩
* **MugenTrainSoundLib**
    * 미설치 시 게임이 충돌할 수 있습니다.

## 수록 내용
* **V1 편성**: 양산 선행차

## 주요 기능
* **실내등**: 전조등 및 후미등과 연동하여 점등됩니다.
* **행선지 표시기**: 실제 차량에 수록된 표시를 모두 포함하고 있습니다.
