## 概要
RealTrainModにJR東日本EV-E301系電車を追加するアドオンパックです。

## 動作確認環境
* Minecraft 1.7.10 (KaizPatchX-1.7.1)
* Minecraft 1.12.2 (RTM 2.4.23-42)

## 導入方法
0. `[RTM]SeriesEV-E301_KarasuyamaLine`を`mods`フォルダから削除する（以前のバージョンが導入されている場合）
1. ダウンロードした`[RTM]Kozu_JR-East_EV-E301Series-Pack.zip`を`mods`にコピーする
2. Minecraftを起動し、各モデルが追加されていることを確認する

### 前提パック
* **MugenTrainSoundLib**
    * 未導入の場合、ゲームがクラッシュする可能性があります。

## 収録内容
* **V1編成**: 量産先行車

## 主な機能
* **室内灯**: 前照灯・尾灯と連動して点灯します。
* **行先表示機**: 実車に収録されている表示を全て収録しています。
