## Overview
This is an addon pack that adds the JR East E501 series train to RealTrainMod.

## Verified Environment
* Minecraft 1.7.10 (KaizPatchX-1.7.1)
* Minecraft 1.12.2 (RTM 2.4.23-42)

## How to Install
0. Delete `[RTM]Kozu_SeriesE501_K703.zip` and `[RTM]Kozu_SeriesE501_K704.zip` from the `mods` folder (if a previous version is installed).
1. Copy the downloaded `[RTM]Kozu_JR-East_E501Series-Pack.zip` to the `mods` folder.
2. Launch Minecraft and confirm that each model has been added.

### Required Packs
* **MugenTrainSoundLib**
    * The game may crash if this is not installed.

## Contents
* **K703 Formation**: Made by Kawasaki Heavy Industries / Current specifications / Single-arm pantograph / LED destination indicator
* **K704 Formation**: Made by Tokyu Car Corporation / Current specifications / Single-arm pantograph / Rollsign destination indicator

## Main Features
* **Interior Lights**: Lights up in conjunction with the headlights and taillights.
* **Train Number Indicator**: You can set a 5-digit train number individually using RTM's custom button function.
